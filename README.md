## ДЗ №1 — Метод дихотомии  
**Задача:** реализовать метод одномерной безусловной минимизации — *метод дихотомии*, согласно индивидуальному варианту (вариант №3).

### Реализовано:
- Веб-приложение на Go  
- Графический интерфейс (HTML/JS + Plotly)  
- Возможность ввода функции f(x)  
- Ввод параметров: a, b, ε, δ, максимум итераций  
- Графическое отображение функции и траектории приближений  
- Таблица итераций  
- SSE-стриминг (итерации в реальном времени)  
- Возможность остановки метода в процессе  
- Экспорт результатов в CSV  
- Отдельная страница со справкой и аналитическим тестом  

### Запуск:
```bash
go mod tidy
go run ./cmd/server
```

После запуска приложение доступно по адресу:
```bash
http://localhost:8080/
```
